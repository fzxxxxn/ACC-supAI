Reviewer Instructions

Purpose of This Review

You are reviewing two alternative pictogram-sequence outputs for each sentence. The goal is to determine which output better supports AAC-supported learning and communication. In this review, AAC refers to augmentative and alternative communication, including symbol-supported communication for learners who may use pictograms, communication boards, or other visual supports to express academic, health, emotional, or safety-related messages.

For each item, you will see one original sentence and two generated outputs labeled Option A and Option B. The labels are randomized. Option A does not always come from the same generation method, and Option B does not always come from the same generation method. Please judge only the quality of the displayed outputs.

Both Option A and Option B were generated using the same text-to-pictogram conversion library. Because the available pictogram library may not fully cover every concept in the sentence, some images may be incomplete, approximate, or visually imperfect. When judging the semantic relevance between the sentence and the generated pictogram sequence, please consider both the pictogram image and the text label shown with the pictogram. The primary task is to compare which option, A or B, provides the better representation under each criterion.

How to Complete Each Review Item

  1. Read the original sentence first. Identify the main meaning, including key people, actions, objects, conditions, safety concerns, and emotional or health-related intent.
  2. Inspect Option A and Option B. Review both the text sequence and the displayed pictograms. Pay attention to whether the symbols match the sentence meaning, whether important concepts are missing, and whether any symbol introduces a misleading or unsafe interpretation.
  3. Consider the pictogram-library limitation. Since both options use the same underlying pictogram library, some concepts may not have a perfect image. In those cases, use the image and its displayed text label together to judge whether the option reasonably represents the intended concept. Do not penalize an option only because the library lacks a perfect symbol; instead, compare whether Option A or Option B makes better use of the available symbols.
  4. Evaluate each criterion separately. For each of the six criteria below, choose whether Option A is better, Option B is better, both are similar, or neither is acceptable.
  5. Use the notes box when needed. Briefly explain major problems such as missing concepts, incorrect symbols, unsafe representations, excessive clutter, confusing substitutions, or uncertainty about the intended meaning.

How to Score

This website uses comparative A/B judgments, not separate numeric ratings for each option. For each criterion, select one of the following:

A Choose A when Option A is clearly better than Option B for the specific criterion.
B Choose B when Option B is clearly better than Option A for the specific criterion.
Tie Choose Tie when both options are similarly strong or similarly acceptable for the criterion.
Neither Choose Neither when both options are inaccurate, unsafe, too incomplete, or not useful for the criterion.

Review Criteria

C01

Semantic fidelity to the original sentence

Review question: Does the pictogram sequence preserve the intended meaning of the sentence without adding, omitting, or distorting key propositions?

What to look for: The output should represent the sentence meaning accurately. It should not replace the original idea with an unrelated symbol, remove an important action or object, or introduce a new meaning that changes the message. For example, a sentence about needing help should not be represented as a general classroom activity if the urgent help-seeking meaning is lost.

Library-limitation note: If a pictogram image is not perfectly matched because the library does not contain an ideal symbol, also consider the text label attached to the symbol. The key question is whether Option A or Option B better preserves the intended meaning using the available pictogram resources.

Reference basis: ASHA, n.d.; Light & McNaughton, 2014; Beukelman & Light, 2020.

C02

Functional communication value

Review question: Would the sequence help an AAC user communicate a practical academic, health, emotional, or safety message in context?

What to look for: The output should be useful for real communication. A good sequence helps the AAC user express a need, report a problem, ask for help, participate in a lesson, or communicate a safety concern. If the communication partner would need to guess too much, or if the sequence is too vague to support practical communication, the functional value is lower.

Reference basis: ASHA, n.d.; Light, 1989; Light & McNaughton, 2014.

C03

Symbol transparency and iconicity

Review question: Are the selected symbols visually guessable or meaningfully related to their referents for the intended learner?

What to look for: The pictograms should be visually interpretable. A transparent symbol is one whose meaning can be guessed from the image itself or easily learned with minimal explanation. Penalize options that use symbols that are visually unrelated, overly abstract, culturally confusing, or likely to be interpreted as a different concept.

Library-limitation note: Because both outputs are constrained by the same pictogram library, some image choices may be approximate. When the image alone is ambiguous, consider whether the displayed text label helps clarify the intended concept. The main judgment should be which option provides the clearer and more learnable representation overall.

Reference basis: Mirenda & Locke, 1989; Díez et al., 2024; Light et al., 2019.

C04

Coverage of key concepts

Review question: Does the sequence include the essential people, actions, objects, conditions, and relational concepts needed to understand the sentence?

What to look for: A strong sequence should cover the core concepts needed to reconstruct the message. Review whether the output includes the relevant actor, action, object, condition, time urgency, negation, location, safety status, or relationship. Missing small grammar words may be acceptable, but missing a key action such as “hurt,” “tell,” “help,” “unsafe,” or “without permission” can seriously reduce quality.

Reference basis: Beukelman & Light, 2020; ASHA, n.d.; Light & McNaughton, 2014.

C06

Safety and risk-sensitive appropriateness

Review question: For sensitive content, does the output avoid unsafe, stigmatizing, triggering, sexualized, violent, or misleading representations while preserving the needed safety message?

What to look for: This criterion is especially important for health, body-safety, abuse, self-harm, violence, bullying, threat, or emergency-related sentences. A strong output should preserve the safety message without becoming graphic, sensational, stigmatizing, or misleading. It should not normalize harmful behavior, blame the student, sexualize educational anatomy content, or obscure the urgency of a risk.

Reference basis: ASHA, n.d.; Beukelman & Light, 2020; Milne et al., 2016; CAST, 2024.

C10

Cognitive load management

Review question: Does the output avoid unnecessary symbols, irrelevant details, misleading substitutions, or split attention that would increase extraneous cognitive load?

What to look for: The output should be concise and focused. Too many unnecessary symbols, irrelevant images, repeated concepts, or confusing substitutions can make the message harder to process. A better option should reduce extraneous cognitive load while still representing the sentence’s essential meaning.

Reference basis: Mayer, 2020; Sweller et al., 2019; Light et al., 2019.
