Reviewer Instructions

For each item, compare Option A and Option B according to the six AAC-supported learning criteria below. For each criterion, select A, B, Tie, or Neither. Your choices are saved locally in this browser and can be exported as a CSV file.

  1. C01 Semantic fidelity: Which option better preserves the intended meaning of the sentence?
  2. C02 Functional communication value: Which option better supports practical AAC communication in context?
  3. C03 Symbol transparency and iconicity: Which option uses more visually guessable and meaningful symbols?
  4. C04 Coverage of key concepts: Which option better includes essential people, actions, objects, conditions, and relations?
  5. C06 Safety and risk-sensitive appropriateness: Which option is safer, more respectful, and less likely to mislead?
  6. C10 Cognitive load management: Which option is more concise, focused, and cognitively efficient?
